Projeto Resilia

Introdução ao Projeto

Bem-vindos à apresentação do nosso projeto de modernização do armazenamento de dados na Resilia!

Neste projeto, nossa equipe se dedicou a reestruturar a forma como a Resilia gerencia informações cruciais relacionadas aos estudantes, facilitadores, turmas, módulos e cursos. O objetivo principal é otimizar a extração de dados estratégicos para facilitar a tomada de decisão.

Nossa jornada envolveu a modelagem do banco de dados, a criação de scripts SQL, a inserção de dados e a resolução de perguntas estratégicas para a empresa. Através deste site, compartilhamos como planejamos e executamos cada etapa do projeto.

Em nossa apresentação, exploraremos os detalhes do projeto, desde a criação da base de dados até a análise exploratória dos resultados. Fique à vontade para baixar os arquivos e scripts do projeto em nosso repositório do GitHub.

Planejamento

Nosso projeto de modernização do armazenamento de dados na Resilia foi cuidadosamente planejado para atender às necessidades estratégicas da empresa. Inicialmente, nos dedicamos a entender profundamente o cenário atual, no qual os dados relacionados a estudantes, facilitadores, turmas, módulos e cursos deveriam ser armazenados, facilitando a análise e tomada de decisões assertivas. Essa análise detalhada nos permitiu identificar e definir objetivos claros para o projeto.

Com os requisitos em mãos, iniciamos a fase de modelagem do banco de dados. Criamos uma estrutura que representa as entidades relevantes e seus atributos, bem como os relacionamentos entre elas. Garantimos que as chaves primárias e estrangeiras fossem definidas corretamente, facilitando a integridade e consistência dos dados.

Diagrama gerado no MySQL Workbench

A seguir, desenvolvemos scripts SQL para criar o banco de dados e suas tabelas, bem como inserir dados de exemplo para testes.

Para atender às necessidades estratégicas da Resilia, também elaboramos consultas SQL que respondem a perguntas específicas sobre a área de ensino. Isso nos permitiu extrair insights valiosos e direcionar ações eficazes com base nos dados.

Além disso, como parte do projeto, pensamos em consultas adicionais que agregassem valor à exploração dos dados. Uma vez que os aspectos técnicos foram concluídos, nos concentramos em preparar uma apresentação detalhada dos resultados. Utilizamos slides para ilustrar nossa modelagem, os principais scripts SQL, os insights obtidos e como atendemos aos requisitos do projeto.

Para enriquecer nossa apresentação e visualizar os insights obtidos, utilizamos o Power BI para criar gráficos intuitivos e informativos. Primeiramente, coletamos os dados do nosso banco de dados MySQL e os importamos para o Power BI. Em seguida, selecionamos as métricas relevantes, como o número de estudantes por turma e o status de evasão, para construir gráficos claros e eficazes.

Gráficos gerados no PowerBI

Utilizamos uma variedade de tipos de gráficos, como barras e pizza, para representar diferentes aspectos dos dados. Com a funcionalidade de filtros interativos, tornamos possível explorar os dados de maneira dinâmica, permitindo que os usuários escolham quais informações desejam visualizar. Além disso, ajustamos cores e rótulos para tornar os gráficos mais legíveis e atraentes.

Ao criar nosso painel no Power BI, destacamos as principais métricas e insights identificados durante o projeto. Isso incluiu a distribuição dos estudantes por turma, a taxa de evasão e a participação de cada facilitador nas aulas. Ao compartilhar esses gráficos na nossa apresentação, conseguimos transmitir de forma clara e impactante os resultados do nosso trabalho e o valor das análises realizadas.

Finalmente, consolidamos nossos esforços em um repositório no GitHub, onde todos os scripts, códigos e materiais da apresentação estão disponíveis para acesso. Essa abordagem colaborativa nos permitiu trabalhar de maneira eficiente como uma equipe coesa e garantir que cada etapa do projeto fosse bem documentada.

Em resumo, nosso planejamento minucioso, dedicação à modelagem de dados, criação de consultas estratégicas e preparação cuidadosa da apresentação resultaram em um projeto sólido que modernizou significativamente a gestão de informações na Resilia.
